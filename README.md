# Platformer

A project whose aim is to auto-generate platformer levels of varying difficulty.

## Status

[![Build Status](https://travis-ci.com/jakalope/platformer.svg?branch=main)](https://travis-ci.com/github/jakalope/platformer)

## Local Build


```
mkdir -p build
cd build
cmake ..
make -j
```

## Run the Tests

```
cd build
make -j && ctest
```
